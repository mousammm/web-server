Technical and digital meanings

    Generated or nonsensical text: In natural language processing (NLP), "rubbish examples" are pieces of text that have been heavily modified to be nonsensical to humans but do not change an AI model's prediction. This is used to test the model's robustness.
    Encoding issues: Text can appear as "rubbish" or a series of random characters if a system is not using the correct character encoding (like UTF-8) to display the language, such as Arabic text not rendering correctly, notes Stack Overflow.
    Poorly formatted text: When pasting text from a source like Microsoft Word, it can include "formatting rubbish" (unwanted, messy formatting) that needs to be cleaned up using a text editor or paste options like "Paste Plain Text".
    Pixelated text: In graphics, "rubbish" can describe jagged or pixelated text, which may be fixed by using anti-aliasing, increasing the canvas size, or changing the file format to one that supports better quality (like PNG over JPEG).
