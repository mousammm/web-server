# C Web Server Implementation 

## Socket Creation
```c
int server_fd = socket(AF_INET, SOCK_STREAM, 0);
```
- **AF_INET**: IPv4 address family
- **SOCK_STREAM**: TCP protocol (reliable, connection-oriented, two-way communication)
- **0**: Default protocol (TCP for SOCK_STREAM)

## Socket Address Initialization
```c
struct sockaddr_in address;
address.sin_family = AF_INET;           // IPv4
address.sin_addr.s_addr = INADDR_ANY;   // Accept connections from any interface
address.sin_port = htons(8080);         // Port 8080 (host to network short)
```
- **INADDR_ANY**: Binds to all available network interfaces
- **htons()**: Converts port number to network byte order (big-endian)

## Binding the Socket
```c
bind(server_fd, (struct sockaddr *)&address, sizeof(address));
```
- **server_fd**: Server socket file descriptor
- **Cast to sockaddr***: `sockaddr_in` is cast to generic `sockaddr` pointer for compatibility
- **sizeof(address)**: Size of the address structure

## Listening for Connections
```c
listen(server_fd, 10);
```
- **server_fd**: Server socket to listen on
- **10**: Backlog queue size (maximum pending connections)

## Accepting Connections
```c
int client_fd = accept(server_fd, 0, 0);
```
- **server_fd**: Listening socket
- **First 0**: NULL client address info (can be filled to get client details)
- **Second 0**: NULL client address length
- Returns: Client socket file descriptor for communication

## Sending HTTP Response
```c
char *response = 
    "HTTP/1.1 200 OK\r\n"
    "Content-Type: text/plain\r\n"
    "Content-Length: 13\r\n"
    "\r\n"
    "Hello, World!";

send(client_fd, response, strlen(response), 0);
```
- **HTTP/1.1 200 OK**: Status line (protocol, status code, reason)
- **Content-Type**: MIME type of response
- **Content-Length**: Size of response body in bytes
- **\r\n\r\n**: Empty line separating headers from body
- **send()**: Transmits data to client socket

## Closing Connections
```c
close(client_fd);
close(server_fd);
```
- **close(client_fd)**: Terminates individual client connection
- **close(server_fd)**: Shuts down the server socket
